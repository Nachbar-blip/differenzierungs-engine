# DifferenzierungsEngine

Adaptives Mathe-Trainingstool für die Oberstufe (Klasse 11/12, GK + LK).

**33 Themen, 1188 Aufgaben** in den Bereichen Analysis, Analytische Geometrie und Stochastik.

## Bedienung

- `index.html` öffnen
- Thema wählen
- Aufgaben lösen — die Engine passt das Schwierigkeitsniveau (Level 1–6) automatisch an
- Fortschritt wird im Browser gespeichert (localStorage)

## Tastatur

- **Enter** — Antwort prüfen
- **→** — nächste Aufgabe
- **T** — Tipp anzeigen
- **1–4** — bei MC-Aufgaben

## Struktur

```
index.html              Startseite
spirale-engine.js       Lern-Logik
spirale.css             Styling
trainer/                33 Themen-HTMLs
```
